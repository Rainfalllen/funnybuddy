🎲 玩法说明

核心：每回合投出若干骰子，把骰子拖/点进装备触发效果——攻击、防御、治疗或施加状态，击败敌人并深入地牢。

装备条件：每件装备只接受满足条件的骰子（任意 / 偶数 / 奇数 / ≤N / ≥N / =N）。先点选一个骰子，再点击高亮的装备即可使用。

状态：☠中毒、🔥燃烧每回合造成持续伤害；❄冰冻把对方最高骰子冻成 1；💧虚弱降低伤害；🌵荆棘被攻击时反弹伤害；🎯易伤受到的伤害提高；🛡护盾抵消伤害（自己回合开始清空）。

敌人意图：你的回合就能看到敌人下回合将造成的伤害/护盾/状态，据此决定进攻还是防御。

大招：受到伤害积累能量，充满后可重掷所有骰子，扭转坏运气。

地牢：每层在战斗 / 精英 / 宝箱 / 商店 / 营火 / ❔事件中择一前进。事件提供风险与收益的抉择。击败本章 Boss 后深入下一章，敌人愈发凶险，直至征服全部章节。
